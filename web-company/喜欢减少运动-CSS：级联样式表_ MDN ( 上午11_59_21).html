重要提示：  本页底部的嵌入式示例的缩放比例移动可能会对某些读者造成问题。患有前庭运动障碍的读者可能希望在观看动画之前在其设备上启用还原运动功能。

句法

no-preference
指示用户未使系统知道首选项。
reduce
表示用户已通知系统他们希望使用一种界面，该界面可以删除或替换引发前庭运动障碍者不适的基于运动的动画类型。

用户偏好

对于Firefox，reduce如果满足以下条件，则可以接受该请求：

  • 在GTK / GNOME中：GNOME调整>常规选项卡（或外观，取决于版本）>动画已关闭。
    • 或者，将其添加gtk-enable-animations = false到GTK 3配置文件的[Settings]块中。
  • 在Windows 10中：设置>轻松访问>显示>在Windows中显示动画。
  • 在Windows 7中：控制面板>易于访问>使计算机更易于查看>关闭所有不必要的动画（如果可能）。
  • 在macOS中：系统偏好设置>辅助功能>显示>降低动作。
  • 在iOS中：“设置”>“常规”>“辅助功能”>“减少运动”。
  • 在Android 9+中：设置>辅助功能>删除动画。
  • 在Firefox中  about:config：添加一个名为的数字首选项ui.prefersReducedMotion，并将其值设置为1。对此首选项所做的更改将立即生效。

例子

默认情况下，此示例具有缩放动画。如果在辅助功能首选项中启用了“减少运动”，则将动画定调为简单的溶解，而无需前庭运动触发器。

的HTML

<div class="animation">animated box</div>

的CSS

.animation {
  animation: pulse 1s linear infinite both; 
}

/* Tone down the animation to avoid vestibular motion triggers like scaling or panning large objects. */
@media (prefers-reduced-motion) {
  .animation {
    animation-name: dissolve;
  }
}
